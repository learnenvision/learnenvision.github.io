BLOOM'S TAXONOMY OF COGNITIVE DEVELOPMENT

The original taxonomy named the different structures based on the nature of the learning task (knowledge, comprehension, application, analysis, synthesis, and evaluation).

KNOWLEDGE (REMEMBERING)

APP FEATURE

In positive reinforcement situations, a response or behavior is strengthened by the addition of praise or a direct reward. In this application if the user will have the highest score among others in a specific game the user's name can be displayed in the leaderboard together with user's highest points.



COMPREHENSION (UNDERSTANDING)

APP FEATURE

These questions test the students’ ability to summarize and describe in their own words without necessarily relating it to anything. In this feature it allows the users to describe or defined the given question.



APPLICATION (TRANSFERING)

APP FEATURE

In positive reinforcement situations, a response or behavior is strengthened by the addition of praise or a direct reward. In this application if the user will have the highest score among others in a specific game the user's name can be displayed in the leaderboard together with user's highest points.



ANALYSIS (RELATING)

APP FEATURE

These questions encourage students to break material into parts, describe patterns and relationships among parts, to subdivide information and to show how it is put together. In this feature the user needs to analyze which is the correct category in each of the instruments.



SYNTHESIS (CREATING)

APP FEATURE

These questions encourage students create something new by using a combination of ideas from different sources to form a new whole. In this feauture the user are allowed to create or develop sound with the illustration given.



EVALUATING (JUDGING)

APP FEATURE

Evaluation questions encourage students to develop opinions and make value decisions about issues based on specific criteria. In this feature it allows the user to rate the experience while using the mobile application.
